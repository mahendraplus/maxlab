1. Operating System (OS) Kya Hai?

  • Computer ko chalane wala main software
  • Hardware aur software ke beech ka bridge
  • Resources ko manage karne wala system

2. OS ke Types

  • Single User OS (jaise Windows)
  • Multi User OS (jaise Linux)
  • Real Time OS (jaise factory machines mein)
  • Batch OS (purane zamane mein use hota tha)
  • Network OS (servers ke liye)

3. Modern OS ki Khaasiyat

  • Multitasking kar sakta hai
  • User friendly interface
  • Security features
  • Resource management
  • Network support

ðŸ”¸ PROCESS MANAGEMENT

1. Process Kya Hai?

  • Program jo execute ho raha hai
  • Har process ka apna ID hota hai (PID)

2. Process States

  • New: Process create hua hai
  • Ready: CPU ke liye wait kar raha hai
  • Running: CPU pe chal raha hai
  • Waiting: Kisi resource ka wait kar raha hai
  • Terminated: Process khatam ho gaya

3. Scheduling Algorithms

  • FCFS: First Come First Serve
  • SJF: Sabse chota kaam pehle
  • Round Robin: Har process ko time slice milta hai
  • Priority: Priority ke hisab se process chalta hai

ðŸ”¸ INTER PROCESS COMMUNICATION (IPC)

1. Critical Section

  • Code ka wo hissa jo shared resources use karta hai
  • Ek time pe sirf ek process access kar sakta hai

2. Mutual Exclusion

  • Ek process critical section mein hai toh doosra nahi ja sakta
  • Deadlock se bachne ke liye zaroori hai

3. Semaphores

  • Binary Semaphore: Traffic signal ki tarah (0 ya 1)
  • Counting Semaphore: Multiple resources ke liye

ðŸ”¸ DEADLOCK

1. Deadlock Kya Hai?

  • Jab processes ek doosre ka wait karte reh jaate hain
  • System hang ho jata hai

2. Deadlock ke 4 Conditions

  • Mutual Exclusion
  • Hold and Wait
  • No Preemption
  • Circular Wait

3. Deadlock Solutions

  • Prevention: Conditions ko rokna
  • Avoidance: Banker's Algorithm
  • Detection: Check karke handle karna

ðŸ”¸ MEMORY MANAGEMENT

1. Paging

  • Memory ko same size ke pages mein divide karna
  • Virtual memory ko physical memory se map karna

2. Virtual Memory

  • Hard disk ko RAM ki tarah use karna
  • Bade programs ko chota RAM mein chalana

3. Page Replacement Algorithms

  • FIFO: First In First Out
  • LRU: Least Recently Used
  • Optimal: Best but practical nahi

ðŸ”¸ FILE MANAGEMENT

1. File System

  • Files ko organize karne ka tarika
  • Directory structure maintain karta hai

2. File Operations

  • Create, Delete, Open, Close
  • Read, Write, Seek
